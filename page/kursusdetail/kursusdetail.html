Apa itu HTML

HTML merupakan singkatan dari HyperText Markup Language, HTML digunakan sebagai pondasi awal dalam pembuatan sebuah website. Pada kerangka HTML terdapat beberapa elemen, tag, serta atribut berikut penjelasannya:

  1. !DOCTYPE html Fungsi dari tag diatas yaitu untuk menyatakan tipe dokumen yang digunakan yaitu html
  2. html lang=”eng” Merupakan tag pembuka dari dokumen html dengan memiliki atribut lang=”en” maksud dari hal tersebut yaitu konten yang digunakan menggunakan bahasa inggris.
  3. head Merupakan bagian kepala dari kerangka html dan dapat di isi dengan tag seperti title salah satunya untuk memberikan judul pada bar address di browser kalian.
  4. body Merupakan bagian badan dari kerangka html dan isi dari konten website kita banyak tag yang dapat digunakan pada tag body salah satunya heading-1.
